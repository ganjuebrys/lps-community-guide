# 线路图与世界地图

本页面提供 LPS 服务器的线路图（线路规划图）和世界地图（游戏世界地图）的相关信息与访问方式。

---

## 线路图

### 什么是线路图？

线路图展示了服务器线路的规划、建设进度和未来发展计划。通过线路图，您可以了解：

- 🚂 线路走向与站点分布
- 📍 重要枢纽与换乘站
- 🔧 建设进度与计划
- 🗺️ 线路编号与命名规则

## 世界地图

### 什么是世界地图？

世界地图是一个交互式的地图界面，展示了游戏世界的地理信息、建筑分布、重要地标等。通过世界地图，您可以：

- 🏛️ 查看服务器中的标志性建筑
- 📍 定位重要地点（如主城、车站、玩家聚集区）
- 🗺️ 了解世界布局与区域划分
- 🔍 搜索特定位置或建筑

### 使用技巧

1. **缩放与平移**：使用鼠标滚轮缩放，拖拽移动视角
2. **搜索功能**：使用搜索框快速定位目标位置
3. **图层切换**：根据需要切换显示不同的图层（如建筑、线路等）
4. **标记收藏**：将常用位置添加到收藏夹，方便快速访问

## 地图更新频率

- **线路图**：通常随新线路开通或重大变更时更新
- **世界地图**：通常每周或每月更新一次，具体取决于服务器规模和更新需求
